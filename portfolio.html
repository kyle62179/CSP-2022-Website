First Project(pltw 1.1.9). Tic Tac Turtle.

For our first Python project me and John Chang decided to create a Tic Tac Toe game. This is a two player game that is coded with Python Turtle graphics which is why we call it Tic Tac Turtle.




Second Project(pltw 1.2.5). Rock Paper Scissors.

For my second Project I worked with Kam and Lucas to create a rock paper scissors game with visuals using Turtle graphics.



In the console the user is asked if they would like to pick rock, paper, or scissors.
After the game is played the user is asked if they would like to play again or quit.


Third Project(pltw 1.3.1). Christmas E-Card.

For my third PLTW prject I worked with Coleman Ng on an interactive Christmas E-Card.



The recipient is given instructions for the card in the console. They can click Rudolf to Receive a short holiday message.
The user can also click on the Santa flying across in his sleigh. This will increase a simple score counter after every click.
